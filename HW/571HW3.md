# Utility Class
-  Class System.Math เพราะว่า class นี้เป็น static ทำให้เราเรียกใช้งานโดยใช้เพียงชื่อ class และตามด้วย method ที่ต้องการได้เลย เช่น
  ```
  Math.PI
  Math.Sqrt(100)
  ``` 
# Mutable Class
-  Class System.Text.StringBuilder เพราะ class นี้ทำให้เราสามารถแก้ไขค่าภายในตัวแปรได้เลยโดยใช้ method ต่างๆ เช่น
  
  ```
  StringBuilder a = new StringBuilder("Hello");
  a.Append("Bro");
  a.Replace("Bro","Boi");
  ```

# Immutable Class
-  Class System.String เพราะเมื่อ String ถูกสร้างมาแล้วจะไม่สามารถเปลี่ยนแปลงค่าได้ เช่น
  ```
  string word = "Something"
  word = word + "Special" //นี้เป็นเพียงการเพิ่ม string เข้าไปไม่ใช่การแก้ไข ถ้าอยากเปลี่ยน string ต้องสร้างตัวใหม่ขึ้นมาเท่านั้น
  ```
